# Google Play API

基于 [facundoolano/google-play-api](https://github.com/facundoolano/google-play-api) 的部署仓库。

## 关于

将 Google Play Scraper 转换为 RESTful API。

## 子模块同步

更新子模块到最新版本：

```bash
git submodule update --remote
```

## 部署

本项目支持 Vercel 一键部署。
